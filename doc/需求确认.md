# **组织领导力诊断系统 - 软件开发需求文档**

## 一、项目背景

为提升企业组织领导力水平，需开发一套**组织领导力诊断系统**，支持对高层、中层、基层管理者进行360度领导力评估，自动生成个人与组织层面的分析报告，辅助企业识别人才优势与发展空间，制定后续发展建议。

---

## 二、系统目标

1. 支持多层级（高层、中层、基层）领导力测评问卷的在线填写与数据收集。
2. 实现自评与他评（上级、平级、下级）数据的自动汇总与计算。
3. 自动生成个人与组织的领导力雷达图与优劣势分析报告。
4. 提供AI辅助解释与行动建议功能。
5. 支持数据导出、报告下载与权限管理。

---

## 三、功能需求

### 3.1 用户管理模块

| 功能 | 描述 |
|------|------|
| 用户注册/登录 | 支持邮箱/手机号注册，角色权限控制（管理员、评估者、被评估者）（先不做有别的team的api） |
| 角色权限 | 不同角色查看不同数据范围（如个人报告、组织报告） |
| 批量导入 | 支持Excel批量导入用户与组织结构 |

---

### 3.2 问卷管理模块

| 功能 | 描述 |
|------|------|
| 问卷配置 | 支持高层、中层、基层三套问卷模板配置 |
| 题目类型 | 单选题、反向计分题、开放题 |
| 计分规则 | 支持正向/反向计分、相关性换算、最终得分计算 |
| 问卷发布 | 支持按组织层级、部门、个人发布问卷 |

---

### 3.3 数据采集模块

| 功能 | 描述 |
|------|------|
| 自评填写 | 被评估者在线填写自评问卷 |
| 他评邀请 | 系统自动发送邀请邮件/短信给上级、平级、下级 |
| 数据校验 | 自动校验填写完整性，提醒未完成项 |

---

### 3.4 数据分析模块

| 功能 | 描述 |
|------|------|
| 得分计算 | 自动计算“直接得分”、“换算分数”、“最终得分” |
| 维度汇总 | 按测评维度（如心智模式、战略与远见等）汇总得分 |
| 对比分析 | 支持自评与他评对比、组织横向对比 |
| 雷达图生成 | 自动生成个人与组织领导力雷达图 |

---

### 3.5 报告生成模块

| 功能 | 描述 |
|------|------|
| 个人报告 | 自动生成包含雷达图、优劣势、AI解释、行动建议的报告 |
| 组织报告 | 自动生成组织整体领导力分析报告 |
| 报告模板 | 支持自定义模板（Word/PDF格式） |
| AI辅助 | 调用AI接口，自动生成优势、劣势解释与发展建议 |

---

### 3.6 系统管理模块

| 功能 | 描述 |
|------|------|
| 日志管理 | 记录用户操作日志、问卷填写日志 |
| 数据备份 | 支持定期自动备份与手动导出 |
| 系统配置 | 支持问卷计分规则、AI接口参数配置 |

---

## 四、非功能需求

| 类别 | 要求 |
|------|------|
| 性能 | 支持100人同时在线填写，报告生成响应时间 < 3秒 |
| 安全 | 数据传输加密，支持权限分级访问，符合GDPR/中国数据合规要求 |
| 可用性 | 系统年可用性 ≥ 99.5%，支持7×24小时运行 |
| 可扩展性 | 支持后续新增测评维度、问卷模板、报告格式 |
| 兼容性 | 支持主流浏览器（Chrome、Edge、Safari），适配移动端 |

---

## 五、技术架构建议

| 层级 | 技术建议 |
|------|----------|
| 前端 | Vue3 + TypeScript，支持响应式布局 |
| 后端 | Node.js（NestJS） |
| 数据库 | PostgreSQL（支持复杂查询与数据分析） |
| 文件存储 | 阿里云OSS（用于报告存储） |
| AI集成 | KimiAPI，生成解释与建议 |
| 部署 | Docker + Kubernetes，支持自动扩容与灰度发布 |
